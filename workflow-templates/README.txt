Bitcoin Puzzle Solver
=====================

Installation:
1. Drag BitcoinPuzzleSolver to Applications folder
2. Launch from Applications
3. Right-click and select "Open" on first launch

Requirements:
- macOS 11.0+
- Python 3.9+
- 4GB RAM

Works on both Intel and Apple Silicon Macs!

For more info: https://github.com/DeadByDawn101/bitcoin-puzzle-solver
